# Smart Glove — ESP32 BLE Wireless Mouse

## What it does
Wearable wireless mouse using ESP32 + MPU-6050.
Controls cursor via hand motion over BLE.

## Components Used
- ESP32
- MPU-6050 IMU
- 2S Li-ion Battery
- Copper tape contacts

## Features
- 3 modes: Air Mouse, 3D Lab, Map Explorer
- BLE HID — no drivers needed
- Presented at Tech-Mania 2K26
